# 更新日志

[中文文档](CHANGELOG.cn.md) | English

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)，
并遵守 [语义化版本](https://semver.org/spec/v2.0.0.html)。

## [3.0.4] - 2025-11-19

### 新增
- 📦 **PyPI 发布**: 现已发布到 PyPI，包名为 `github-pr-analyzer`。
- 📅 **遍历命令**: 新增 `traverse` 命令，用于批量分析近期 PR。
- 🛠️ **简化安装**: 支持通过 `pip install` 安装。
- ⚡ **入口点**: 新增 `gh-pr-analyzer` CLI 命令。

## [2.0.0] - 2025-01-21

### 新增
- 🔍 **AI 智能搜索**: 使用 cursor-agent 的智能关键词提取
- 📅 **增强的日期时间显示**: 全面的日期时间信息
- ⚙️ **环境变量配置**: 简化的设置方式

## [1.0.0] - 2025-01-21

### 新增
- GitHub PR Analyzer 初始版本
- PR 和 Commit 收集
- 智能模糊搜索
- Diff 查看器
- AI 驱动分析
